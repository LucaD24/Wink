# Static Site Template

This is a minimal static site starter that pulls all copy from `content.json`.

## Customize

1. Update `content.json` to change the title, text, and call to action.
2. Edit `styles.css` to change colors and layout.
3. Update `index.html` if you want to add new sections.

## Local Preview

```sh
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
