# Tools

## new_project.js

Scaffold a new project from the static site template.

```sh
node tools/new_project.js 2026_03_14 anniversary_note
```

This creates:

```
projects/2026_03_14_anniversary_note/
  index.html
  styles.css
  script.js
  content.json
  assets/
    images/README.md
    audio/README.md
  README.md
```
